# 💻 Loan Management System

## 📌 Description

A full-stack Loan Management System developed using Java Spring Boot and HTML, CSS, JavaScript.
This system allows users to apply for loans and admins to manage approvals and reports.

---

## 🚀 Features

* User Registration & Login
* Loan Application Submission
* Admin Approval / Rejection
* EMI Calculation
* Loan Status Tracking
* Reports & Dashboard

---

## 🛠️ Tech Stack

* Backend: Java Spring Boot
* Frontend: HTML, CSS, JavaScript
* Database: PostgreSQL
* Build Tool: Maven

---

## ▶️ How to Run

1. Clone the repository
2. Open project in IDE
3. Run:

   ```
   mvn spring-boot:run
   ```
4. Open browser:

   ```
   http://localhost:8080
   ```

---

## 📂 Project Structure

* `src/main/java` → Backend code
* `src/main/resources/static` → Frontend files
* `pom.xml` → Maven configuration
